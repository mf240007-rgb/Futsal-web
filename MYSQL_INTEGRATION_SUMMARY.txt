╔═══════════════════════════════════════════════════════════════════════════╗
║                        TROUBLESHOOTING                                    ║
╚═══════════════════════════════════════════════════════════════════════════╝

❌ Error: Communications link failure
   ➜ MySQL di XAMPP belum running
   ➜ Solution: Start MySQL dari XAMPP Control Panel

❌ Error: Access denied for user 'root'
   ➜ Password MySQL tidak match
   ➜ Solution: Update application.properties

❌ Database tidak ter-create
   ➜ Parameter createDatabaseIfNotExist tidak bekerja
   ➜ Solution: Buat manual di phpMyAdmin atau run database_setup.sql

❌ Port 3306 already in use
   ➜ Ada MySQL lain yang running
   ➜ Solution: Stop MySQL lain atau ubah port

❌ Tables tidak ter-create
   ➜ DDL auto tidak bekerja
   ➜ Solution: Check console logs, verify ddl-auto=update

Baca MYSQL_SETUP_GUIDE.md untuk troubleshooting lengkap!
